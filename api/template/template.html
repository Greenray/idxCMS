File   classes/template.class.php
Overview   Completely separated from php code. Handles special tags and replaces them with commands of the php interpreter. It is able recursively execute functions and directives: FOREACH, IF, ELSE, ELSEIF, SWITCH, CASE, BREAK, DEFAULT, INCLUDE, CONTINUE, ISSET, EMPTY, EXIT, to use constants and variables. There is a possibility of caching the result.
